syntax:
<meta name=""  content="">
//no closing tag 
//name and content is called attribute in html 
//attribute extra information about that particular tag.
//any programming if you want to string you need to use either 
'' single quote or "" double quote 

example:
<meta name="Keywords" content="Laptop,Mobile,Desktop">
<meta name="Description" content="But online e-commerce,
50%offer on laptop">
<meta name="author" content="flipkart">
<meta name="viewport" content="device=device-width,initial-scale=1">
<meta charset="utf-8">
//character encoding html
